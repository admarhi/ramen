# Get a cluster from a `ConsortiaMetabolismSet` Object

Gets a cluster from a `ConsortiumMetabolismSet` object.

## Usage

``` r
getCluster(object, node_id, name = NA_character_, description = NA_character_)

# S4 method for class 'ConsortiumMetabolismSet'
getCluster(object, node_id, name = NA_character_, description = NA_character_)
```

## Arguments

- object:

  A `ConsortiumMetabolismSet` object

- node_id:

  Numeric scalar giving the node to be extracted.

- name:

  Character scalar specifying a name for the selection.

- description:

  Character scalar describing the selection.

## Value

A cluster from a `ConsortiumMetabolismSet` object
