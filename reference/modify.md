# Modify a `ConsortiaMetabolismSet` Object

Modifies a `ConsortiumMetabolismSet` by adding or removing
`ConsortiumMetabolism` objects. This allows for dynamic updating of
consortium sets for comparative analyses.

## Usage

``` r
modify(object)

# S4 method for class 'ConsortiumMetabolismSet'
modify(object)
```

## Arguments

- object:

  An object of class `ConsortiumMetabolismSet`

## Value

A modified `ConsortiumMetabolismSet` object

The modified `ConsortiumMetabolismSet` object.

## Methods (by class)

- `modify(ConsortiumMetabolismSet)`: Add or Remove Consortia to/from a
  `ConsortiumMetaboliteSet` Object
