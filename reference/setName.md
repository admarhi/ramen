# Set Names or description of ramen Objects

Sets a new name or description of `ConsortiumMetabolism`,
`ConsortiumMetabolismSet`, or `ConsortiumMetabolismAlignment` objects

## Usage

``` r
setName(object, value = NA_character_)

# S4 method for class 'ConsortiumMetabolism'
setName(object, value = NA_character_)

# S4 method for class 'ConsortiumMetabolismSet'
setName(object, value = NA_character_)

# S4 method for class 'ConsortiumMetabolismAlignment'
setName(object, value = NA_character_)
```

## Arguments

- object:

  A `ConsortiumMetabolism`, `ConsortiumMetabolismSet`, or
  `ConsortiumMetabolismAlignment` object

- value:

  Character scalar specifying a name.

## Value

A cluster from a `ConsortiumMetabolismSet` object
